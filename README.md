## Cambios principales

### Estructura global y estilos base

- Importadas las fuentes **DM Sans**, **Inter** y **Poppins** desde Google Fonts.
- Incorporada la librería **Bootstrap Icons** para íconos sociales y de interfaz.
- Creada una paleta completa de **variables CSS (`:root`)** para colores neutrales, primarios y secundarios.
- Aplicado un **reset global** con `box-sizing: border-box` y se estableció **DM Sans** como fuente base.

---

### Navegación (Navbar)

- Implementada una **barra sticky**

- Añadido efecto **hover** en enlaces y botón de usuario.
- Incorporado **icono de menú hamburguesa** visible en pantallas menores.
- Añadida una versión **responsive del menú desplegable**.
